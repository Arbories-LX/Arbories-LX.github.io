关于Tomcat Address Localhost报错的问题

近期频繁使用Tomcat，今天运行的时候报错tomcat address localhost：1099 is already in use

表示1099端口已经被占用

解决方法：

  • 管理员身份运行命令提示符，我获取更高的权限对系统进行设置
  • 输入netstat -ano|findstr 1099
  • 输入taskkill -f -pid 5904

成功解决
